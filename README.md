### project Install
폴더별로 node_modules가 생성 되어야합니다.<br/> 
vsCode에서 터미널을 띄워 아래처럼 진행<br/> 
터미널 단축키 : ctrl + `

1.폴더로 이동
```
cd directory
```
2.node_modules 설치
```
npm i
```

sample
```
cd day01
npm i
```

### project start
node_modules가 있는 프로젝트에서 실행시켜야합니다.

```
npm start
```

### 새 프로잭트 생성
- 생성
```
npx create-react-app 프로젝트명
```
- 생성 프로젝트로 이동
```
cd 프로젝트명
```
- 실행
```
npm start
```
[참고사이트 바로가기](https://create-react-app.dev/docs/getting-started)
